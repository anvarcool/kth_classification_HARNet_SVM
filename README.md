# kth_classification_HARNet_SVM
Данное упражнение - упрощенный вариант исследования о HARNet-SVM \ В нем взята только архитектура модели и несложная часть предобработки. Сложные техники не использовались, так как они не описаны должным образом в статье
\ Удалось добиться точности 45% на тестовом множестве. Неизвестно каким образом авторы получили высокую точность.
\ В моей реализации обучающее и тестовое множество разделялись таким образом, чтобы объекты съемки (люди) в этих множествах не пересекались.
\ Возможно, авторы не учитывали, что нейронной сети проще предсказать действие, совершенное человеком, которого она ранее видела.
