# kth_classification_HARNet_SVM
Данное упражнение - упрощенный вариант исследования о HARNet-SVM на наборе данных KTH  
В нем взята только архитектура модели и несложная часть предобработки. Сложные техники не использовались, так как они не описаны должным образом в статье  
Удалось добиться точности 45% на тестовом множестве. Неизвестно каким образом авторы получили высокую точность.  
В моей реализации обучающее и тестовое множество разделялись таким образом, чтобы объекты съемки (люди) в этих множествах не пересекались.  
Возможно, авторы не учитывали, что нейронной сети проще предсказать действие, совершенное человеком, которого она ранее видела.

# Файлы
video_to_tensors.ipynb - извлечение кадров видео и сохранение в виде torch.tensor  
network_train.ipynb - обучение нейронной сети  
svm_classification.ipynb - классификация с помощью SVC на основе выходного вектора из нейронной сети  
train.csv - обучающая выборка (выходной вектор из нейронной сети на обучающем множестве)  
test.csv - тестовая выборка (выходной вектор из нейронной сети на тестовом множестве)
